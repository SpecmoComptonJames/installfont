Installfont

Super easy system font installer for Nodejs

Download .zip Download .tar.gz View on GitHub

Installation

npm install installfont

Quick Usage

var installfont = require('installfont');

installfont('path/to/your/font.ttf', function(err) {
  if(err) console.log(err, err.stack);;
  //handle callback tasks here
});

This module is not tightly coupled to any specific font libraries or apis. It also is not opinionated about how you get the fonts on your machine.

Compatible OS:

  • Mac OS
  • Windows
